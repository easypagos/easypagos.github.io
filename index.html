API simple y elegante para una integración plug'n'play.

Con una simple llamada a nuestra API REST comienza el proceso de pago de tu cliente.



  >>> import requests

  >>> response = requests.post('https://api.easypagos.cl', {
          'amount': '29456', 
          'order_id': 'o1234', 
          'success_url': 'http://example.net/success', 
          'failure_url': 'http://example.net/failure'
      })

  >>> response.json()['redirect_url']
  u'http://pagos.easypagos.cl'
